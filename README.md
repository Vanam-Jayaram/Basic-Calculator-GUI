# Basic Calculator GUI
A MATLAB GUI application that performs basic arithmetic operations through an interactive graphical interface.
## Features
* Addition
* Subtraction
* Multiplication
* Division
* Easy-to-use graphical interface
## Technologies Used
* MATLAB
* MATLAB GUI (.fig)
## Files
* `calculator.m` - Application logic
* `calculator.fig` - GUI layout
## How to Run
1. Open MATLAB.
2. Open the project folder.
3. Open `calculator.fig` or run `calculator.m`.
4. Use the calculator interface to perform arithmetic operations.
